What is RevSpace

RevSpace is a web app that helps you make a calendar that when imported to Google calendar, helps schedule revision dates for spaced repetition


What is spaced repetition

To know what spaced repetition method is, lets learn about something first.
When you learn something, you tend to forget it after a few days, i.e. it decays, starting from an hour after you've learned it.

The above image shows how huch information you retain over time (unless you are a genius who remembers everything).
As you can see, you forget what you learned over time.
To combat this, there are a lot of memorization or retention techniques one of which, is Spaced Repetition.

Spaced Repetition helps you remember what you learned for a longer time

How spaced repetition works

Spaced Repetition involves having the learner actively recall what you learned from memory at gradually increasing intervals.

At each interval, after recalling, review your performance, see what you missed and then add more information(what you missed and also what you didn't)

This graph shows how after each iteration or after each recall, information is retained for a longer period, but the next review happens before your memory of the information decays significantly.

After a few repetitions (iterations) the curve flattens out, i.e. you remember it for long time.

How are the intervals scheduled

Now there are quite a few methods like

  1. SM-2

    2. Revise on 1, 7, 16, and 35 days after learning

  2. i111 method

    3. Revise immediately after, 1 day, 1 week and then 1 month after learning

  3. My method

    4. This is a combination method that is useful for both easy and hard topics.
    It has 9 iterations.

    1. Immediately after
    2. 1 day later
    3. 7 days later
    4. 14 days later
    5. 21 days later
    6. 30 days later
    7. 60 days later
    8. 120 days later
    9. 180 days later
    Its not too steep, though it might feel slow for some.

How this web app works?

This app, takes your topics with starting date and time with your timezone.
It then takes this data, calculates dates for next revisions and creates an ICS(Internet Calendaring and Scheduling) file which you can import into your calendars.
When imported, your revisions are scheduled into calendars instead of doing it manually

p.s. None of your date is collected or stored, everything happens on your side during your session. Deleting the list or reloading the window will erase everything.


Guide:
  1. Timezone

    2. Your timezone is detected automatically, but if you want to choose another one just check the Timezone list, change it and press Set Timezone button

  2. Topics

    3. These are the actual topics you want to revise,
    input the Topic, its starting date is present day by default(which you can change) and set the starting time.

    To add more topics, press Add More Topics button

    To add all topics to the list, press Add All Topics button

  3. Topics list

    4. This is a table that displays all your topics with its title, starting date and dates for the upcoming revisions.

    At the end of each row, there is a delete icon whuch is visible when you hover over it.

    Below the list is the Delete List button which deletes the entire list

  4. Download ICS

    5. This button when pressed, initiates the download of ICS file

  5. Add to Google Calendar

    6. This button opens the google calendar import/export page in a new tab where you can import your ICS file.

    Or you can do this manually in the calendar you want if it has a feature to import ics files.
